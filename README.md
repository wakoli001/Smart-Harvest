# Smart-Harvest
----A program to create a post harvest monitor system.The program checks the conditions of humidity and temperatures, It then compares them with the safe threshold set within the system .The program perfoms its logic by the 
help of the logics set in the system.There is a notification sent to the manager of the storeee in case the error detection system is activated by the system .An alarm\buzzer is also activated by the system in case the error detection system is on. The alarm buzzes and the led bulbs are turned on.
In case the condition is normal, the green bulb is lit whreas when the condition is unwanted the red bulb i sturned on.
